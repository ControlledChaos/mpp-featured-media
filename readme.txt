== Description ==

An add-on for MediaPress. Allow users to mark media as featured and list featured media using "( MediaPress ) Featured Media" widget.

== Installation ==

This section describes how to install the plugin and get it working.

You may need to follow the following steps.

1. Download `mpp-featured-media.xyz.zip` (where xyz is version number )and extract
2. Upload `mpp-featured-media` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's all.
